# 诸暨话语料采集授权同意书模板

本同意书用于诸暨话开放语料库采集语音、转写文本和相关匿名元数据。请在录音前向发音人说明清楚，并由发音人选择授权等级。

## 基本说明

我已了解：

- 本次录音将用于诸暨话的保存、整理、研究和展示。
- 我可以选择匿名，项目不会公开我的真实姓名、手机号、身份证号、详细住址等敏感信息。
- 我可以选择是否公开原始音频。
- 我可以选择是否允许资料用于 AI 训练。
- 我可以选择是否允许商业使用。
- 我可以联系项目维护者，要求在未来版本中移除我的资料。已经被他人下载或引用的历史版本，项目方会尽力说明和更新，但可能无法完全追回。

## 授权等级

请勾选一项：

- [ ] A. 仅用于文化保存，不公开原始音频。可保存匿名元数据和必要说明。
- [ ] B. 可公开音频与转写，但不可商业使用。
- [ ] C. 可公开音频与转写，并允许研究、教育和 AI 训练使用。

是否允许商业使用：

- [ ] 允许
- [ ] 不允许

是否允许 AI 训练：

- [ ] 允许
- [ ] 不允许

是否选择匿名：

- [ ] 是
- [ ] 否
