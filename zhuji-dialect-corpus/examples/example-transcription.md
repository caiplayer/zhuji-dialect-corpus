# 示例转写

## 原始诸暨话转写

```text
S001: 今朝天气蛮好个，吃好饭出去走走。
S002: 嗯，老底子这辰光，街上人多得来…[笑]
S001: 我记得（阿婆/阿太）以前常讲，过年要早点准备米果。
```

## 普通话释义

```text
S001: 今天天气很好，吃完饭出去走走。
S002: 是啊，以前这个时候，街上的人很多……[笑]
S001: 我记得奶奶或曾祖母以前常说，过年要早点准备米果。
```

## 备注

- `(阿婆/阿太)` 表示转写者不确定听到的是哪一个词。
- `[笑]` 表示发音人笑了。
- `…` 表示停顿。
- 如果完全听不清，应标为 `[听不清]`，不要猜写。
