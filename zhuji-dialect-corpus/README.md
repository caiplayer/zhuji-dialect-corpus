# 诸暨话开放语料库

**项目名 / Project:** `zhuji-dialect-corpus`  
**中文名:** 诸暨话开放语料库  
**当前版本:** v0.1

## 项目简介

诸暨话开放语料库是一个面向浙江诸暨话的开放资料项目，用于采集、整理、保存诸暨话的语音、转写文本、普通话释义和基础元数据。

This project is an open corpus initiative for the Zhuji dialect of Zhejiang, China. It aims to preserve speech and text materials for cultural documentation, linguistic research, dialect ASR, TTS, and community memory.

## 为什么要保存诸暨话

诸暨话承载着地方生活、亲属称谓、风俗、地名、行业经验和一代人的说话方式。随着人口流动、普通话普及和年轻一代语言习惯变化，很多词、音、句式和讲故事的方式正在变少。保存诸暨话，不是为了把它固定成一种唯一标准，而是为了让真实的差异被看见、被研究、被后人听到。

## 项目目标

- 建立规范、可持续、可扩展的诸暨话开放语料库结构。
- 保存不同乡镇、年龄层和语境中的诸暨话材料。
- 支持方言保护、语言学研究、ASR 方言识别、TTS 语音合成和地方文化保存。
- 提供采集、授权、转写、元数据和隐私伦理的基础规范。

## 数据类型

- 语音录音：词表、句表、自由讲述、自然对话、民俗故事等。
- 文本转写：贴近诸暨话实际说法的汉字转写。
- 普通话释义：帮助非本地读者理解内容。
- 规范化文本：供机器学习、检索和数据处理使用。
- 元数据：发音人匿名信息、地区、年龄组、录音设备、授权等级等。

## 如何参与

1. 阅读 `docs/collection-guide.md` 和 `docs/transcription-guide.md`。
2. 在录音前取得发音人明确授权，并记录授权等级。
3. 按命名规范采集音频，填写 `data/speakers.csv`、`data/recordings.csv`、`data/segments.csv`。
4. 按三层转写方式补充 `dialect_text`、`mandarin_translation`、`normalized_text`。
5. 运行 `python scripts/validate_metadata.py` 检查字段。
6. 提交 Pull Request，由维护者审核。

本项目不接受无授权来源的音频、视频、微信群转发内容或其他第三方材料。

## 隐私和授权原则

- 文本、转写和元数据默认建议采用 **CC BY-NC 4.0**。
- 代码脚本建议采用 **MIT License**，便于复用和改进。
- 音频材料的授权以发音人签署或确认的同意书为准。
- 不要未经授权上传他人音频、视频、短视频、微信群转发内容或家庭私密录音。
- 默认匿名处理，不公开身份证号、手机号、详细住址、真实姓名等敏感信息。
- 发音人可以选择是否公开音频、是否允许 AI 训练、是否允许商业使用。

## 免责声明

本项目不是“标准化诸暨话”，也不试图规定哪一种说法才是“正确”的诸暨话。诸暨话内部存在地域、年龄、家庭、迁移经历和使用场景的差异。本项目的目标是记录这些差异，而不是抹平它们。
