# 转写规范

本项目采用三层转写，既保留诸暨话的实际表达，也方便检索和机器学习使用。

## 三层转写

### dialect_text

尽量贴近诸暨话实际说法的汉字转写。可以保留语气词、重复、停顿和口音差异。不要为了“标准化”而删除本地说法。

### mandarin_translation

用普通话解释大意。可以不逐字对应，但应忠实表达原意。

### normalized_text

供机器学习和检索使用的规范化文本。可以去掉明显口吃、重复和非语义噪声，但不要改写意思。

## 标注规则

- 不确定的字词可以用括号标注，例如：`我（侬/恁）个晓得`。
- 听不清用 `[听不清]`。
- 笑声用 `[笑]`。
- 明显停顿用 `…`。
- 多人说话必须标出 `speaker_id`。
- 不要为了让文本看起来像普通话而删除口音差异、方言词和本地语序。

## 多人对话示例

```text
S001: 今朝天气蛮好个。
S002: 是个，出去走走蛮适意。
```

## 审核建议

- 第一遍转写保留听到的内容。
- 第二遍补普通话释义。
- 第三遍检查时间戳、发音人编号和不确定标注。
