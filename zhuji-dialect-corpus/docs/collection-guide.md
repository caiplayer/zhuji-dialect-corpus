# 采集说明

本说明用于帮助志愿者采集清晰、可追溯、可授权的诸暨话资料。手机录音也可以，关键是环境安静、授权明确、元数据完整。

## 录音环境

- 尽量选择安静房间，减少回声。
- 不要开电视、风扇、音乐、洗衣机等持续噪声源。
- 手机可以使用，但请尽量靠近发音人，距离约 20 到 40 厘米。
- 录音前先录 3 秒环境底噪，方便后期判断噪声情况。
- 如果多人对话，请尽量让两人距离设备相近。

## 推荐格式

- 推荐 WAV 或 FLAC。
- 推荐采样率 44.1kHz 或 48kHz。
- 如只能使用手机默认格式，也可以先提交，但请在 `recordings.csv` 中写明设备和格式。

## 文件命名规则

示例：

```text
ZJ001_S001_20260429_free_001.wav
```

含义：

- `ZJ001`：录音编号或采集批次编号。
- `S001`：发音人匿名编号。
- `20260429`：录音日期，格式为 YYYYMMDD。
- `free`：录音类型。可用值包括 `wordlist`、`sentence`、`free`、`conversation`、`story`、`culture`。
- `001`：同一批次中的序号。

## 推荐采集内容

- 基础词表：身体、亲属、数字、天气、饮食、动作、日常物品等。
- 固定句表：问候、买菜、问路、家庭、天气、责备、惊讶、劝说等。
- 自由讲述：小时候生活、过年、婚礼、上学、做生意、种田等。
- 自然对话：两个本地人围绕熟悉话题自由聊天。
- 民俗故事：西施传说、地方故事、村里老说法。
- 童谣、俗语、歇后语、老话。
- 地名、亲属称谓、行业词汇。

## 采集记录

每次录音后，请及时填写：

- `speakers.csv`：发音人的匿名信息。
- `recordings.csv`：整段录音的信息。
- `segments.csv`：切分后的片段、时间戳和转写。

不要等到很多天后再补，记忆会很快变模糊。
